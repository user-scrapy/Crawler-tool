# Crawler-tool
Built purely in Python, this project allows users to scrape target websites via commands. Each module and feature comes with documentation detailing usage and syntax. This is its initial form.

# Features
-Static web page scraping (requests + BS)
-Dynamic web page scraping (playwright)
-Modular command system for easy extension
-Lightweight - runs with minimal configuration
